## 🧬 About the Project

> 🚀 A machine learning project that predicts whether a person will buy health insurance. Built with **streamlit**, **pandas**, **scikit-learn**, and wrapped in a modern web interface.

## 🚀 How to Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/Dheeraj23-08/Health_insurance_predictor_1.git
cd Health_insurance_predictor_1

# 2. Create a virtual environment (optional)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the app
streamlit run app.py
